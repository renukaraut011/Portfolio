PCB & IOT Module - 7 Days

Day 1: IoT Module: Getting Started with ESP8266

Topics Covered:

Introduction to ESP8266
• Microcontroller features: Built-in Wi-Fi, GPIO pins, low-power operation.
• Pinout explanation and typical IoT applications (smart home devices, sensors, automation).

Arduino IDE Installation & Setup
• Step-by-step installation of Arduino IDE.
• Adding ESP8266 board manager URL and downloading required libraries.

First Hands-on Programs
• Blink Internal LED: Wrote and uploaded a basic program to blink the onboard LED.
• Blink External LED: Interfaced an external LED with ESP8266 GPIO pins and controlled it through code.

Key Features Learned:
✅ Understanding of microcontroller architecture.
✅ Basics of embedded programming using Arduino IDE.
✅ Practical circuit connection and testing.

Day 2: IoT Module: Wi-Fi & Web Server Basics

Topics Covered:

Networking Concepts for IoT
• Role of Wi-Fi in IoT.
• Basics of IP addresses, SSID, and secure password handling.

ESP8266 Wi-Fi Setup
• Connecting ESP8266 to a Wi-Fi network and verifying connectivity.

Practical Programs
• Program 1 – Wi-Fi Connection: Printed the module’s IP address in Serial Monitor to confirm successful connection.
• Program 2 – Simple Web Server: Hosted a minimal webpage on ESP8266 to control LED ON/OFF directly from a browser.

Key Features Learned:
✅ Concept of IoT devices communicating over Wi-Fi.
✅ Basics of creating and hosting a web server on a microcontroller.
✅ Remote control of hardware through a web interface.

Day 3: IoT Module: Sensor Integration & Real-Time Monitoring

Topics Covered:

Quick Recap of Wi-Fi connection and web server setup.

Ultrasonic Sensor Interfacing

Practical Programs
• Program 1 – Distance Measurement: Measured distance and displayed output on Serial Monitor.
• Program 2 – Live Web Page: Displayed real-time sensor readings directly on a web page accessible via Wi-Fi.

Key Features Learned:
✅ Interfacing sensors with microcontrollers.
✅ Real-time data acquisition and live monitoring.
✅ Understanding how IoT makes remote sensing possible.
