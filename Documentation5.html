AI/ML Module - 7 Days

Day 1: Python for Data Science

Goal: Build the programming foundation needed for AI/ML.

Python is the backbone of modern data science because it is easy to read and has a huge ecosystem of libraries.

Key Concepts & Practice:

Understand variables, data types and how Python handles numbers and strings.
Work with lists, dictionaries, and sets to store and manipulate data efficiently.
Control program flow using loops and conditional statements.
Learn how to write and reuse functions to keep code organised.
Read and write data from text or CSV files so you can bring real datasets into Python.
Outcome: You should feel confident writing small scripts and understand why Python is the first choice for data work.

Day 2: Numerical Computing with Numpy

Goal: Perform high-speed mathematical operations.

NumPy provides multi-dimensional arrays and is far faster than plain Python lists.

Key Concepts & Practice:

Create and reshape arrays; slice and index data efficiently.
Use vectorised operations and broadcasting to replace slow loops.
Apply built-in statistical functions—mean, median, standard deviation—to large datasets.
Understand why almost every ML library (like TensorFlow or scikit-learn) depends on NumPy internally.
Outcome: Comfortably handle large numerical datasets and perform quick mathematical analysis.

Day 3: Data Analysis with Pandas

Goal: Learn to clean and explore real-world data.

Pandas makes tabular data manipulation straightforward.

Key Concepts & Practice:

Differentiate between Series and DataFrames, and when to use each.
Import messy CSV/Excel files, remove duplicates, and handle missing values.
Transform and filter data using Boolean indexing.
Group and aggregate data to create meaningful summaries.
Merge or join multiple datasets for richer analysis.
Outcome: Ability to prepare raw data into a structured form ready for visualization or machine learning.

Day 4: Visualizing Data

Goal: Turn numbers into stories.

Before modelling, data must be understood visually.

Key Concepts & Practice:

Create common charts—bar graphs, histograms, scatter plots, and boxplots.
Recognise when to use each type of chart to highlight relationships or distributions.
Customise colours, labels, legends and layouts to make plots publication-ready.
Detect trends, outliers and patterns that numbers alone may hide.
Outcome: You can present insights visually and make data more accessible to non-technical audiences.

Day 5: Machine Learning Basics

Goal: Understand how machines find patterns and make predictions.

Machine learning is about teaching computers to learn from data without being explicitly programmed.

Key Concepts & Practice:

Distinguish between supervised learning (predicting known outcomes) and unsupervised learning (finding hidden patterns).
Recognise the difference between features (inputs) and labels (outputs).
Split data into training and testing sets to measure real-world performance.
Get an overview of classic algorithms: linear regression for continuous data, decision trees for classification.
Learn evaluation metrics such as accuracy, precision, recall and F1 score.
Outcome: A clear picture of the ML pipeline and the basic terminology used in the field

Day 6: Building & Tuning Models

Goal: Put theory into practice.

Here you actually fit models and see how different settings affect performance.

Key Concepts & Practice:

Train simple models on well-known datasets such as Iris or Titanic.
Perform hyperparameter tuning to improve predictions and avoid overfitting.
Apply cross-validation to check model stability.
Learn to interpret metrics and identify when a model is underfitting or overfitting.
Outcome: You can confidently build a basic ML model, test it, and iterate to improve accuracy..

Day 7: Next-Step

Goal: Move beyond tutorials toward practical, production-ready skills.

Key Concepts & Practice:

Prepare larger, more complex datasets: handle outliers, scale features, and create pipelines.
Introduction to ensemble methods (like random forests and boosting) and why combining models often gives better results.
High-level overview of deep learning and how neural networks differ from classical ML.
Explore open datasets or Kaggle competitions to practise on real projects.
Learn about deploying models on cloud platforms or integrating them into simple applications.
Outcome: A roadmap for continuing your AI/ML journey and applying your skills in internships, research or personal projects.
